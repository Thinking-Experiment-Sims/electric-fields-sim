Lab Procedure
  1. Select a Case (I, II, or III)
  2. Hover canvas — probe reads voltage
  3. Find 4–5 points at the same voltage (e.g. 5 V)
  4. Record each — write coordinates on your paper
  5. Connect the dots → that's an equipotential line!
  6. Repeat for 1 V, 2 V, 3 V, 4 V
  7. Draw field lines perpendicular to equipotentials

Key Equations
ΔV = VB − VA
Wfield = −q·ΔV
Work done by field on charge q moving A→B
ΔKE = Wfield (Work-Energy Theorem)
ΔPE = −Wfield (Conservative field)
W > 0 → spontaneous (field pushes)
W < 0 → needs external work (push)
W = 0 → equipotential path, no energy change
